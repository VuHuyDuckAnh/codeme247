css overflow 
        The CSS overflow property controls what happens to content that is too big to fit into an area.
        Thuộc tính overflow kiểm soát điều gì xảy ra khi nội dung bên trong thẻ html mà quá lớn 
        không vừa thẻ html chứa nội dung đó 

        overflow: visible; ( luôn hiển thị ) là giá trị mặc định 
        overflow: hidden; ( ẩn nội dung vượt quá kích cỡ của thẻ html )
        overflow: auto; ( hiển thị thanh scrollbar khi nội dung tràn ra bên ngoài và
        không hiển thị thanh scrollbar khi nội dung không tràn ra bên ngoài )
        overflow: scroll; ( luôn hiển thị scrollbar bất kể nội dung có tràn ra bên ngoài hay không)

overflow: visible;
